# 接口

## 获取班级列表class/getList

同样应当注意权限，对于老师和助教只返回所在的班级

请求POST

```json
{
    token:"..."
}
```

响应

```json
{
    success,
    info,
    data:{
        id,
        name,
        time:{
            semester,
            term,
        }
    }
}
```



## 获取班级成员列表class/getMemberList

请求POST

```json
{
    token:"...",
    class:"class ID"
}
```

响应

```json
{
    success,
    info,
    data:{
        teacher:[users],
		assistant:[],
     	student:[]
    }
}
class user{
    username,
	id,
	name,
	access,
	class:[]
}
```



## 批量添加班级成员class/addMemberList

请求POST

```json
{
    token:"...",
    class:"class id",
    member:{
        teacher:[users],//和user/addList一样需要判断用户是否已经存在，考虑一下合并还是覆盖
        assistant:[],
        student:[]
    }
}
```



## 批量添加班级class/addList

请求POST

```json
{
    token,
    class:[classes]
}
```



## 批量删除班级class/delList

```json
{
    token,
    class:[classID]
}
```



## 批量修改班级class/configList

```json
{
    token,
    class:[classes]
}
```
